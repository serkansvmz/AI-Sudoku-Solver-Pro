# 🧩 AI Sudoku Solver & Game

Bu proje; görüntü işleme, derin öğrenme ve klasik algoritmaları bir araya getiren kapsamlı bir Sudoku uygulamasıdır. Kullanıcılar fotoğraftaki bir Sudoku'yu saniyeler içinde çözdürebilir veya kendileri oyun oynayabilir.

## 🚀 Özellikler
- **AI Rakam Tanıma:** PyTorch tabanlı CNN modeli ile yüksek doğrulukta rakam tespiti.
- **Görüntü İşleme:** OpenCV kullanarak perspektif düzeltme ve ızgara ayıklama.
- **İnteraktif Oyun:** Farklı zorluk seviyelerinde Sudoku üretme, gerçek zamanlı hata kontrolü (Kırmızı/Yeşil) ve canlı kronometre.
- **Backtracking Algoritması:** 9x9 boyutunda her türlü Sudoku'yu saniyeler içinde çözen algoritma.

## 🛠️ Kurulum
1. Depoyu bilgisayarınıza indirin:
   `git clone https://github.com/kullaniciadi/AI-Sudoku-Solver-Pro.git`
2. Gerekli kütüphaneleri kurun:
   `pip install -r requirements.txt`
3. Uygulamayı başlatın:
   `python src/main.py`
